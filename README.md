# MCDA
Multicriteria Decision Aid

This is a java based software that solves the following MCDA problems: 

* Electre I, 
* Electre I_s, 
* Electre I_v, 
* Electre II, 
* Electre III, 
* Electre IV, 
* Electre TRI and 
* Electre TRI ME.

Download J-Electre-v1.0 at: https://sourceforge.net/projects/j-electre/files/
